# Music Files

Place your MP3 file here with the name: `hymn-for-the-weekend.mp3`

The music player will automatically load and play this file when visitors come to your website.

**To add the song:**
1. Download "Hymn for the Weekend" by Coldplay as an MP3 file
2. Rename it to `hymn-for-the-weekend.mp3`
3. Place it in this folder (`public/music/`)

The file path in the code is: `/music/hymn-for-the-weekend.mp3`
